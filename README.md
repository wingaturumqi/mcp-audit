# mcp-audit

基于 **T/TAF 352—2026**《模型上下文协议服务器（MCP Server）安全技术要求》的合规分级自查 CLI。

## 功能

- 🔍 自动发现 MCP 配置（Claude Desktop、Cursor、VS Code、Windsurf、.mcp.json）
- 📊 基于 T/TAF 352 标准的 L1/L2/L3 三级合规评估
- 📋 64 条检查项，覆盖 7 大安全维度
- 💡 每条检查项附带整改建议

## 安装

```bash
go install github.com/wingaturumqi/mcp-audit@latest
```

或下载对应平台的二进制文件。

## 使用

```bash
# 扫描所有 MCP 配置
mcp-audit scan

# 查看帮助
mcp-audit --help
```

## T/TAF 352 安全维度

| 维度 | 检查项数 |
|------|---------|
| 5.1 身份鉴别与访问控制 | 7 |
| 5.2 通信与接口安全 | 10 |
| 5.3 工具与资源安全 | 12 |
| 5.4 执行环境安全 | 12 |
| 5.5 供应链与更新安全 | 9 |
| 5.6 数据安全与隐私保护 | 8 |
| 5.7 日志审计与安全监控 | 6 |
| **合计** | **64** |

## 合规等级

- **L1 基础级**（21 项）— 所有 MCP Server 应满足的基本安全要求
- **L2 增强级**（47 项）— 面向企业级部署的增强安全能力
- **L3 高级级**（64 项）— 最高等级，含运行时防护和高级隔离
